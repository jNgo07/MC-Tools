Brief Instructions For Use
1. Copy start-occt.bat and the /OCCT/ folder to a flash drive.
2. Insert flash drive with the above files in to the system you are testing.
3. Run start-occt.bat.

This will copy all necessary files from the flash drive into a newly created OCCT folder on the current user Desktop. You may remove the flash drive after all files have copied.

Without any interaction, the 32 minute OCCT stability test will automatically run within 30 seconds.
Pressing any key during the 30 seconds will open a menu allowing you to run a specific phase of the test.
